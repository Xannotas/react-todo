# React Todo app
Демо: https://xannotas.github.io/react-todo
### Стэк:
  + ReactJS
  + Redux
  + Typescript

### Задачи:
+ Добавление папок под задачи и выбор цвета
+ Переключение между папок и возможность посмотреть все задачи
+ Переименование и удаление папок
+ Добавление и удаление задач
